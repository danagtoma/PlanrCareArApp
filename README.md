## General Information
AR Plant Care Assistant is a mobile augmented reality application developed in Unity using C#. The app helps users understand and care for their houseplants by providing real-time information directly overlaid onto the physical plant. Using image detection, the app identifies plants and displays plant-specific care information.
The goal of the application is to make plant-care information more accessible by visualising invisible data, such as plant health, watering needs and environmental conditions.

## Features:
* Real-time Plant Detection - Identifies plants using image detection.
* AR Information Overlay - Displays care information directly on the plant.
* Health & Condition Insights - Visual feedback about plant health and potential issues.
* Actionable Recommendations - Guidance on watering, nutrition, sunlight and required actions.
* Interactive AR Elements - Users can tap AR objects to access more detailed information.

## Technologies Used
- Unity
- C#
- AR Foundation
- ARCore / ARKit
